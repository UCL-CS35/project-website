Requirements

We gather the project requirements through our discussions with our client, Dr Skipper and analysing similar project like NeuroVault and Neurosynth.

As individuals who don't have any knowledge on neuroscience, we also used ourselves as the target users and identify requirements that will be important to the non-technical users like us.

The following list of requirements had been identified by the team and approved by our client, Dr. Skipper.

Use Cases

We have written a list of basic and crticial use cases for our project. This list is subject to change during the development process.

ID Use Case
1 Register an Account
2 Login
3 Logout
4 View Movie
5 View Term for a specific Movie
6 View Term for all Movies
7 View Component
8 Upload Raw Dataset
9 View Uploaded Raw Dataset Informationi
10 Edit Uploaded Raw Dataset Information
Use Case 1: Register an Account
Primary Actors Neuroscientist User
Secondary Actors None
Preconditions Actor not logged in, does not have an account
Main Flow 1. Click "Register" from navigation bar
2. Enter personal information
3. Click "Submit" button
Post Conditions New account registered, logged in, return to "My Collections" page
Alternative Flows Failed to register new account, error message(s) shown on same page
A) Duplicate email.
B) Field(s) left blank.
Use Case 2: Login
Primary Actors Neuroscientist User
Secondary Actors None
Preconditions Actor not logged in, owns an account
Main Flow 1. Click "Login" from navigation bar
2. Enter Login information
3. Click "Login" button
Post Conditions Actor logged in, return to "My Collections" page
Alternative Flows Invalid Login, error message shown on same page
"Invalid Login, please login using registered email address and password, or register"
Use Case 3: Logout
Primary Actors Neuroscientist User
Secondary Actors None
Preconditions Actor logged in
Main Flow 1. Click "User" on navigation bar
2. Click "Logout" from drop down menu
Post Conditions Return to home page, logged out
Alternative Flows None
Use Case 4: View Movie
Primary Actors Neuroscientist User, Public User
Secondary Actors None
Preconditions None
Main Flow Select Movie
1. From "latest Movies" list on Home page
2. OR from list on "Movies" page
Post Conditions View page for selected Movie
Alternative Flows None
Use Case 5: View Term for a specific Movie
Primary Actors Neuroscientist User, Public User
Secondary Actors None
Preconditions None
Main Flow 1. View Movie
2. From list of Terms under the "Terms" tab, select a Term
Post Conditions On page for selected Term in selected Movie
Alternative Flows None
Use Case 6: View Term for all Movies
Primary Actors Neuroscientist User, Public User
Secondary Actors None
Preconditions None
Main Flow 1. Select "Terms" tab on navigation bar
2. Select a Term from list of Terms
OR
1. View Term for Specific Movie
2. Click "View (term) in all Movies"
Post Conditions On page for selected term in all Movies
Alternative Flows None
Use Case 7: View Component
Primary Actors Neuroscientist User, Public User
Secondary Actors None
Preconditions None
Main Flow 1. View Movie
2. From list of Components under "Components" tab, select a Component
OR
1. View Term (either for specific Movie or all Movies)
2. From list of Components under "Components" tab, select a Component
Post Conditions On page for selected Component
Alternative Flows None
Use Case 8: Upload Raw Dataset
Primary Actors Neuroscientist User
Secondary Actors None
Preconditions Logged In
Main Flow 1. Click "Contribute" tab on navigation bar
2. Click "Choose file" button
3. Choose File(s) from computer
4. Fill in fields for information about contribution
5. Click "Submit" button
6. Read through disclaimer / terms & condition, click "Confirm"
Post Conditions New Dataset uploaded, pending analysis
Alternative Flows 1. Data not uploaded
2. Return to "Contribute" page
Use Case 9: View Uploaded Raw Dataset Information
Primary Actors Neuroscientist User
Secondary Actors None
Preconditions Logged In, Uploaded Dataset previously
Main Flow 1. Click (user's name) tab
2. From drop-down menu click "My Collections"
3. From list of uploaded Dataset, select one
Post Conditions On page with information about uploaded Dataset
Alternative Flows None
Use Case 10: Edit Uploaded Raw Dataset Information
Primary Actors Neuroscientist User
Secondary Actors None
Preconditions Logged In, Uploaded Dataset previously
Main Flow 1. View Uploaded Raw Dataset Information
2. Click "Edit Information"
3. On editing page, update any fields that requires new information
4. Click "Submit"
Post Conditions On page with updated information about uploaded Dataset
Alternative Flows Return to information page without updated information

Glossary

We build a Glossary of Terms to define vocublary for the project so that the team and the client uses the same definition during the discussion of the project.

Term Definition
Term The keyword that describe the brain activation.
Component An analysed image that contain the x-y-z axis of a dataset.
Repository or INcDb The web application that will be publically available.
Processed Dataset The set of files that are produced after our client has run the analysis process on the Raw Dataset.
Raw Dataset The set of brain images collected by researcher when someone watch a movie.
