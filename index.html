NEUROCINEMATICS

Systems Engineering Project : Team 35

Introduction

The objective of this project is to build a social data repository for the Internet Neurocinematics Database (INcDb), which is a database of functional magnetic resource imaging (fMRI) data collected while people watch full length movies.

The data repository will allow users to browse and interact with the data. Scientists will also be able to contribute their film watching fMRI data to the repository.

The team will be developing the data repository using the open-source code associated with http://www.neurosynth.org (available at https://github.com/neurosynth/).

Features of INcDb

Discover

Discover how our brain reacts while watching movies

Contribute

Allow scientists to contribute their data and make discovery of the study

Educate

Use as education tools for human neuroscience
